# HTML / CSS 基礎

## 🎯 目的
Webページの構造と装飾の基本を理解し、自力で静的なページを組み立てられるようにする。

## 🎯 今週のゴール
- HTMLの基本構造を理解し、自己紹介ページを作成できる
- よく使うHTMLタグとCSSプロパティを使いこなせる
- flex-boxで要素を横並びにするなど基本的なレイアウト調整ができる
