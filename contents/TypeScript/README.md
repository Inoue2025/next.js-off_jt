## ✅ 演習①：型の注釈

- `types-training.ts` を作成し、以下の変数に型注釈を追加してみましょう：

```ts
const name = "Hanako";
const age = 22;
const fruits = ["りんご", "バナナ", "みかん"];

function getProfile(user) {
  return `${user.name} (${user.age}歳)`;
}
```

```tsx
const name: string = "Yuki";

export default function Home() {
  return (
    <main>
      <h1>こんにちは、{name}さん</h1>
    </main>
  );
}
```
